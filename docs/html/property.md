### 内容类型
取消过去输入联想
```html
<input type="text" name="title" value="" disableautocomplete/>
```

禁止自动填写表单
```html
<input type="text" name="title" value="" autocomplete="off"/>
```

语音输入
```html
<input type="text" x-webkit-speech lang="zh-CN"/>
```

可开启该元素的编辑模式
```html
<div contentEditable='true'></div>
```
