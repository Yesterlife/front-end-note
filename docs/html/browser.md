### 切换至兼容IE7模式
```html
<meta http-equiv="X-UA-Compatible" content="IE=EmulateIE7"/>
```

### 切换至IE8
```html
<meta http-equiv="X-UA-Compatible" content="IE=8"/>
```

### 360浏览器内核切换
极速核
```html
<meta name="renderer" content="webkit">
```

IE兼容内核
```html
<meta name="renderer" content="ie-comp">
```

IE标准内核
```html
<meta name="renderer" content="ie-stand">
```
